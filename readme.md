## 🧭 About · О проекте

Personal portfolio of **Nikita Ip** (Chris). I currently build with JavaScript and I'm moving into **Python and neural networks** — so the whole site is dressed in the visual language of deep learning: layers, signals, activations and the **viridis** colormap. Built from scratch with vanilla HTML, CSS and JS. No frameworks, no build step.

> Личное портфолио **Никиты Ип** (Chris). Сейчас пишу на JavaScript и ухожу в **Python и нейросети**, поэтому весь сайт оформлен в визуальном языке глубокого обучения: слои, сигналы, активации и палитра **viridis**. Сделано с нуля на чистых HTML, CSS и JS. Без фреймворков и сборки.

- 🗂 **Tabbed projects** — JavaScript *(shipped)* vs Python *(in development)*
